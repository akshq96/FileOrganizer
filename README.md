# RustChef

A Rust project.

## Getting Started

### Prerequisites

- Rust (install from [rustup.rs](https://rustup.rs/))

### Building

```bash
cargo build
```

### Running

```bash
cargo run
```

### Testing

```bash
cargo test
```
